# MUS Score App

Application web simple pour compter les points au jeu de cartes basque **Mus**.

## Fonctionnalités

- Comptage des **ttantto** et **hamarreko**
- Conversion automatique : 5 ttantto = 1 hamarreko
- Détection de victoire (8 hamarreko)
- Animation de victoire avec confettis 🎉
- Noms d’équipes modifiables
- Interface responsive (mobile, tablette, desktop)

## En cours
